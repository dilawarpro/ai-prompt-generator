About AI Prompt Generator

What is AI Prompt Generator?

AI Prompt Generator is a tool designed to help you create effective prompts for AI systems like ChatGPT, DALL-E, and other AI models. A well-crafted prompt can significantly improve the quality of AI-generated content.

User Manual

How to Use This App

  1. Select the Text tab on the home page.
  2. Enter your content in the text area.
  3. Choose a template from the Templates page or use the default Creative Writing template.
  4. Adjust options like formality, detail level, creativity, and instruction clarity as needed.
  5. Click "Generate Prompt" to create your AI prompt.
  6. You can copy, edit, or save the generated prompt using the buttons below it.

  1. Select the Image tab on the home page.
  2. Upload an image using the file selector.
  3. Add context (optional) to provide additional information about the image.
  4. Click "Generate Image Prompt" to create a prompt describing the image.
  5. The app will analyze the image filename and generate a description.
  6. You can copy, edit, or save the generated prompt using the buttons below it.

The app offers several templates to structure your prompts:

  • Creative Writing: For creative and engaging content with vivid language.
  • Academic: For well-researched, academic analysis with relevant theories.
  • Business: For professional business responses with actionable insights.
  • Technical: For detailed technical explanations with implementation details.
  • Image Description: For describing images with visual elements and composition.

To use a template:

  1. Go to the Templates page from the navbar.
  2. Click on a template to select it.
  3. The app will automatically switch to the appropriate tab and apply the template.

The app saves your generated prompts for future reference:

  1. Click Save on a generated prompt to add it to your history.
  2. Access your saved prompts from the History page.
  3. For each history item, you can:
    • Load: Bring the prompt back to the generator.
    • Edit: Modify the title or content of the saved prompt.
    • Delete: Remove the prompt from your history.
  4. Use Clear History to remove all saved prompts.

Customize the app behavior in the Settings page:

  • Dark Mode: Toggle between light and dark themes.
  • Save History: Enable or disable saving prompts to history.
  • History Limit: Set the maximum number of history items to keep.
  • Default Options: Set default values for formality, detail, creativity, and instruction clarity.

Click Save Settings to apply your changes.

Tips for Effective Prompts
  • Be specific: Clearly state what you want the AI to do.
  • Provide context: Give background information when necessary.
  • Set constraints: Specify length, format, tone, or style requirements.
  • Use examples: Provide examples of the desired output when possible.
  • Iterate: Refine your prompts based on the AI's responses.
